# Phishing Prevention Checklist

## 1. Awareness and Training
- Conduct phishing awareness training  
- Run simulated phishing tests  
- Keep updated on phishing threats  

## 2. Email Security
- Verify sender email addresses  
- Hover over links before clicking  
- Avoid unexpected attachments  
- Use spam and malware filters  
- Enable DMARC, SPF, DKIM  

## 3. Passwords and Accounts
- Use strong, unique passwords  
- Enable multi-factor authentication (MFA)  
- Monitor account activity  

## 4. Web and Network
- Check website security (HTTPS/SSL)  
- Avoid suspicious or shortened links  
- Keep browsers updated  
- Block known malicious websites  

## 5. Device Security
- Keep OS and apps updated  
- Install antivirus/anti-malware  
- Avoid public Wi-Fi; use VPN  

## 6. Incident Response
- Report suspicious emails  
- Do not provide credentials  
- Isolate and scan devices if compromised  
- Change passwords immediately if affected  

## 7. Policy and Governance
- Implement security policies  
- Audit and review logs regularly  
- Maintain offline backups  

## Daily Reminders
- Verify sender emails  
- Hover before clicking links  
- Avoid unknown attachments  
- Keep software updated  
- Use MFA  
- Report suspicious emails  
- Review passwords regularly
