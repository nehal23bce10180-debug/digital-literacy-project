## 📌 Project Description
This project was completed as part of the Digital Literacy course. The objective of this project is to develop essential digital skills and promote awareness about responsible and safe use of digital technologies. 

The project consists of five tasks that cover various aspects such as infographic creation, building a digital portfolio, exploring coding platforms, understanding email etiquette, and learning about cybercrime awareness. Each task focuses on practical learning and real-world application of digital tools.
